# caravan-rpc-macros — Pre-release placeholder

Procedural macros for the [Caravan](https://github.com/paulxiep/caravan) RPC SDK. Used internally by the `caravan-rpc` crate; not intended to be depended on directly.

**Status: pre-release placeholder.** Exports `#[wagon]` as an identity attribute macro so SDK-wrapped trait declarations compile cleanly while the functional adapter codegen is being designed (M2 per the [development plan](https://github.com/paulxiep/caravan/blob/main/docs/development_plan.md)).
